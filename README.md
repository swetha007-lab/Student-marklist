# Student-marklist
" project that generates a student marksheet based on the marks entered by the user. It calculates the average marks of three subjects and determines the grade according to the average score. The project is easy to understand for beginners and demonstrates the use of input handling, conditional statements, and basic arithmetic operations in Python.

⚙️ Features

 Takes student name and marks for 3 subjects as input
Calculates the average of the marks

Determines the grade based on average marks

Displays a formatted marksheet output
